---PROGRAM MUST BE RUN ON LINUX/MAC---

---DIRECTIONS FOR USE---

---COMPILE INSTRUCTIONS---
open terminal in the directory this readme is located.

Run 'make' to build the application
Run 'make p2' to buld only the part2 executable
Run 'make p3' to build only the part3 executable
Run 'make p4' to build only the part4 executable
Run 'make clean' to delete the executables before rebuilding

---RUNNING JUST THE PROGRAM---
open terminal in the directory in which the executable files are located and
run the following command.

Usage for part2:  ./p2

Usage for part3:  ./p3

Usage for part4:  ./p4 [subpart]

[subpart] is optional.

Options for subpart:
    1 - show results for 4.1
    2 - show results for 4.2
    3 - show results for 4.3
    4 - show results for 4.4
